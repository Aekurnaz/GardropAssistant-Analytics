# 📊 GardropAssistant: AI Wardrobe Analytics Dashboard
**End-to-End Power BI & SQL Data Analytics Portfolio Project for an AI App.**

Welcome! This repository contains an end-to-end data analytics portfolio project built for a fictional AI-powered Wardrobe Assistant application. 

The primary goal of this project is to showcase advanced data modeling, DAX proficiency, Cohort & Churn analysis, and actionable product intelligence.

## 📂 Repository Structure & How to Explore

To make evaluating this project as easy as possible, everything is neatly organized:

* 📄 **[GardropAssistant_Project_Presentation.pdf](https://github.com/Aekurnaz/GardropAssistant-Analytics/blob/main/Analytics%20%26%20Growth%20Dashboard_%20AI%20Wardrobe%20Assistant.pdf)** 👉 **START HERE.** A detailed, page-by-page visual presentation. It explains the business logic ("The Story") and the technical implementation ("How I did it") for every KPI and chart.

* 📊 **[Download Interactive Power BI Dashboard (Google Drive)](https://drive.google.com/file/d/1N168mCCwcxe_tJ7vAXuc3GH5yRb9bdhS/view?usp=sharing)** The fully interactive `.pbix` file. (Note: External link due to file size). Explore the underlying data model and DAX measures.

* 💻 **[GardropAssistant_Project.sql](https://github.com/Aekurnaz/GardropAssistant-Analytics/blob/main/GardropAssistant_Project.sql)** Contains the backend SQL scripts for RFM (Recency, Frequency, Monetary) segmentation using Window Functions and CTEs.

## 🛠️ Tech Stack & Skills
* **Power BI & DAX:** Interactive Dashboards, Bookmarks, Filter Contexts, Time Intelligence, and Cohort Calculations.
* **SQL (SSMS):** Views, Aggregate Functions, and Window Functions created via SQL Server Management Studio.
* **Product Analytics:** Conversion Funnels, User Retention (Churn Rate), AI Success Metrics, and more.

📌 **Note: The dataset used in this project is entirely synthetic and generated via AI for demonstration and portfolio purposes. It does not represent any real company, users, or actual business data.**
